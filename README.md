# kvmtop

## Description
KVM performance monitoring tool providing real-time metrics, process lists, and resource utilization.

## Build Instructions

### Prerequisites
- GCC or Clang
- Make

### Building
```bash
make
```

The output binary `kvmtop` will be created in the `build/` directory.

## Usage
```bash
./build/kvmtop
```
